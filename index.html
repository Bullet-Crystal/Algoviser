About This Website

This website showcases interactive visualizations of pathfinding algorithms, highlighting the step-by-step exploration of nodes and optimal routes.

Create Maze: Generates a random maze layout to test algorithmic performance and behavior.

Dijkstra: Explores all nodes evenly to guarantee the shortest path. Ideal for weighted grids without heuristics.

DFS (Depth-First Search): Dives deep into paths before backtracking. Fast but doesn‚Äôt guarantee the shortest path.

BFS (Breadth-First Search): Explores neighbors in order of proximity. Always finds the shortest path in unweighted grids.

My Custom: A handcrafted algorithm that combines depth-first search with distance-based sorting, prioritizing nodes closer to the goal for a smart and dynamic pathfinding experience.

A*: Efficient and accurate, using both actual cost and estimated distance to find the optimal path quickly.
